Welcome to the Interfaccia Tutorial

This repository is designed to streamline the process of creating a modular, multi-device dashboard with simulated interactivity while minimizing Python code. It features functions that automate the creation of charts and tables, and it provides automatic download links for CSV files, allowing users to download data without the need for server storage. Additionally, it addresses the issue of resizing Plotly charts when display is set to none. Check out the pages below to explore its various features and capabilities.

▼

INSTRUCTION

This project consists of two main scripts and a script for creating the dashboard:

- "intervaccia.py" : Automates various processes for creating the dashboard.
- "template_interfaccia.html" : Contains the structure of the dashboard.
- "dashboard_esempio.py" : Provides an example code for generating the dashboard.

To ensure the code functions properly, you should install the following libraries:
    - jinja2 3.1.4,
    - plotly 5.23.0,
    - pandas 2.2.2,
    - numpy 2.0.1.

To call the "interfaccia.py" file, keep it in the same folder as the other three files.
Alternatively, if you are working in a notebook, you can simply copy the contents of
the "interfaccia.py" file to the top of the notebook file where you will create your dashboard.

CODE

Importing Libraries

The code begins by importing essential libraries:

Plotly: Used to create interactive charts, which are then converted to HTML.

Jinja2: Handles HTML templates, allowing for dynamic customization of pages.

Numpy and Pandas: Crucial for data manipulation and generation, these libraries provide the data used in charts and tables.

Automatic Graph Creation with `automatic`

Function `create_column`: This function is designed to quickly set up complex graphs, accepting parameters like columns for axes, chart type (e.g., `scatter`, `bar`, `area`, `pie`), and the option to add secondary axes for more detailed visualization.

Function `automatic`: This is the core function for generating graphs. It automatically transforms Pandas DataFrames into interactive HTML charts and tables. Simply specify basic configurations, such as chart type or columns to display, and the function takes care of the rest. The graphs are made interactive through Plotly and are embedded directly into the dashboard as HTML code.

Menu Display with `display`

Function `display`: This function manages the visibility of menu items in the dashboard. It ensures the first menu item is visible while hiding others, making it easier to navigate between different sections of the dashboard. This function is integrated with JavaScript code in the HTML template, adding interactivity to the menus and allowing users to smoothly switch between pages.

Dashboard Generation with `dashboard`

Function `dashboard`: This is the main function that oversees the entire dashboard creation process. Using Jinja2, it loads a predefined HTML template and populates it with the data and graphs generated earlier. You can directly insert HTML code into different sections of the dashboard, allowing for complete customization of page layouts. Additionally, thanks to a JavaScript connection in the template, the dashboard includes download buttons that let users download the chart data as CSV files directly from the browser, without storing the data on a server.

Page Structure and Layouts

Navigation Menu Structure (`navbar`): Includes the brand name, a subtitle, and a series of menu items, each with potential submenus. This menu is dynamic and can be customized via HTML.

Page Types (`home` and `pages`): The system provides two main types of pages, each with various layouts that can be applied. These layouts allow you to organize page content differently, depending on your specific project needs. Whether displaying a single chart or arranging a series of tables and images, the available layouts enable you to create a visually cohesive and functional dashboard.
